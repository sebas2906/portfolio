Agent Core

Agent Core is a project based con Nest, Typescript, MongoDB, LangChain and Cloud services served with docker containers. The purpose of this project is to provide a simple server that can be customized with multiple tools to provide any functionality. Agent Core iniciatilices a new AI Agent based on current configurations in MongoDB. This allows to create different agents for multiple purposes (e.g. Scheduling calendars, Analizing databases, create orders, etc...) for different types of business.
